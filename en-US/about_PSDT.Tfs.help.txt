        about_PSDT.Tfs
        
SHORT DESCRIPTION
        PSDT.Tfs is a collection of PowerShell cmdlets, which improves developer productivity.

LONG DESCRIPTION
        The PowerShell Developer Tools for Team Foundation Server simplifies working with
        Team Foundation Server version control.
        
        PSDT.Tfs is shipped with a built-in (nested) module, which is intended to encapsulate 
        general PSDT application logic.
         
        The following functionality is supported:
        
        * Getting of latest sources
